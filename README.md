# Hello XIAO Cam to RX/TX

> I bought eight Seeed Studio XIAO ESP32-S3 Sense boards to work with the KS4036 Micro:bit kit.
> ESP32-S3 Sense, [B0C69FFVHH](https://www.amazon.com.be/dp/B0C69FFVHH?ref=ppx_yo2ov_dt_b_fed_asin_title)

What I need is an ESP32 connected to an SSD1306 display that can relay bytes not used by RX/TX to another 3.7-volt device.

So I need to explore the possibility of:

* using RX/TX from the ESP32 to the Micro:bit
* using the webcam in the Quest 3 or Android over HTTP
* testing the possibility of using the SSD1306 and webcam on the same Wi-Fi network
